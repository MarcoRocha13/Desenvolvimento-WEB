CSS

Cascading Style Sheets ou Folhas de Estilo em Cascata tem por objetivo realizar a estilização de conteúdo de documentos HTML. Podem definir cores, fontes, espaçamento dentre outros.

Componentes

Têm-se a seguinte sintaxe para a linguagem CSS: 

             seletor 
             {
                 propriedades: valor; /*comentário
                 propriedades-subpropriedades: valor:
             }
  • Selector: seleciona elementos HTML com base no nome do elemento ao qual o estilo está aplicado.
  • Propriedades: O aspecto especifico do elemento que você quer usar como estilo.
  • Valores: configuração aplicada ao atributo.

Há três formas de utilizar o CSS:

Inline

Os códigos CSS são definidos dentro de uma tag HTML. Afetam somente a tag em que foi definido.

Incorporado

Os código CSS são definidos no cabeçalho, dentro da tag <head>. Afetam somente o documento em que foi definido.

Externo

Os código CSS são definidos em um arquivo externo. Afetam todos os documentos que estão linkados ao CSS.
