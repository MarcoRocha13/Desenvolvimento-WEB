Branches

São ramificações de um projeto de software ou documento e tem como objetivo o desenvolvimento paralelo de:

  • Novas funcionalidades;
  • Correção de problemas;
  • Alteração de código.

Quando têm-se um projeto, esse pode ser alocado:

  • Branch Master ou Main: Braço principal de um projeto em um repositório./L1>
  • Branch Secundária: Braço secundário de um projeto em um repositório.

Hospedagem

O GitHub Pages oferece hospedagem gratuita de sites. Basta criar uma conta e um repositório público e a URL será https://nomeusuario.github.io/repositorio.

Para hospedar o seu site através do GitHub Pages realiza as etapas a seguir:

Acesse o seu repositório e clique em configurações.

Role a página até Páginas do GitHub.

Por fim, o seu site será hopstado no URL gerado pelo próprio GitHub.
