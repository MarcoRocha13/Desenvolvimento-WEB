Incorporar vídeos ao site através da máquina local

Segue um video explicativo de como utilizar tag <video>. A tag é usada para incorporar conteúdo de vídeo em um documento, como um clipe de filme ou outras transmissões de vídeo. A tag contém uma ou mais tags com diferentes fontes de vídeo. O navegador escolherá a primeira fonte que ele suporta.

Existem três formatos de vídeo suportados em HTML: MP4, WebM e OGG.

O comando controls especifica que os controles de vídeo devem ser exibidos (como um botão de reprodução/pausa etc).

O comando poster especifica uma imagem a ser mostrada enquanto o vídeo está baixando, ou até que o usuário aperte o botão de reprodução

O comando básico para inserir um vídeo está descrito abaixo:

          <video> width="320" height="240" poster="img" controls>
             source src="movie.mp4" type="video/mp4"
             source src="movie.ogg" type="video/ogg"
          </video>
