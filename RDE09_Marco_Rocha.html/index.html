Incorporar vídeos ao site através da máquina local

Segue um video explicativo de como utilizar tag <video>. A tag é usada para incorporar conteúdo de vídeo em um documento, como um clipe de filme ou outras transmissões de vídeo. A tag contém uma ou mais tags com diferentes fontes de vídeo. O navegador escolherá a primeira fonte que ele suporta.

Existem três formatos de vídeo suportados em HTML: MP4, WebM e OGG.

O comando controls especifica que os controles de vídeo devem ser exibidos (como um botão de reprodução/pausa etc).

O comando poster especifica uma imagem a ser mostrada enquanto o vídeo está baixando, ou até que o usuário aperte o botão de reprodução

O comando básico para inserir um vídeo está descrito abaixo:


          <video> width="320" height="240" poster="img" controls>
            <source src="movie.mp4" type="video/mp4">
            <source src="movie.ogg" type="video/ogg">
          </video>

Incorporar audio ao site armazenados na máquina local máquina local

Para inserir um arquivo de audio utiliza-se a tag <audio>. A tag é usada para incorporar conteúdo sonoro em um documento, como música ou outros fluxos de áudio. A tag contém uma ou mais tags com diferentes fontes de áudio. O navegador escolherá a primeira fonte que ele suporta.

Existem três formatos de áudio suportados em HTML: MP3, WAV e OGG.

O comando controls especifica que os controles de vídeo devem ser exibidos (como um botão de reprodução/pausa etc).

O comando preload especifica se e como o autor acha que o áudio deve ser carregado quando a página é carregada

O comando src especifica a URL do arquivo de áudio.

A seguir está estrutura básica de como se utilizar a tag <audio> e um video explicativo: 


        <audio> preload="metadata/auto/none" controls>
           <source src="audio.mp4" type="audio/mp4">
           <source src="audio.ogg" type="audio/ogg">
        </audio>

Incorporar imagens com a tag <picture>

A tag dá aos desenvolvedores web mais flexibilidade na especificação de recursos de imagem. O uso mais comum do elemento será para direção de arte em designs responsivos. Em vez de ter uma imagem que é dimensionada para cima ou para baixo com base na largura do viewport, várias imagens podem ser projetadas para preencher melhor a configuração do navegador.

O navegador procurará o primeiro elemento <source> onde a consulta de mídia corresponda à largura de viewport atual e, em seguida, exibirá a imagem adequada (especificada no atributo srcset). O elemento <img> é necessário como o último filho do elemento, como opção de recuo se nenhuma das tags de origem corresponder.

Observação: O elemento funciona "semelhante" a <video> e <audio>. Você configura diferentes fontes, e a primeira fonte que se encaixa nas preferências é a que está sendo usada.

A seguir está estrutura básica de como se utilizar a tag <picture> e um video explicativo: 


         <picture>
            <source media="(max-width:tamanho máximo da tela em pixels)" srcset="img.formato">
            <source media="(max-width:tamanho máximo da tela em pixels)" srcset="img.formato">
            <img src="img.formato" alt="descrição">
         </picture>
