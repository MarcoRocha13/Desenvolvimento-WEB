Tag <table>

A tag define uma tabela HTML.

Uma tabela HTML consiste em um elemento e um ou mais <tr>, <th> e <td> elementos. O elemento <tr> define uma linha de tabela, o elemento <th> define um cabeçalho de mesa e o elemento <td> define uma célula de mesa. A tag <table> aceita tanto atributos globais quanto atributos de evento.

Uma tabela HTML também pode incluir <caption>, <colgroup>, <thead>, <tfood> e <tbody> elementos.

A seguir está um exemplo de tabela e em seguência a estrutura básica sem uso de CSS.

Marco marco@gmail.com 99806-4749

Tag <sub>

A tag define texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e, às vezes, é renderizado em uma fonte menor. Texto subscrito pode ser usado para fórmulas químicas.

Exemplo:H2O

Tag <sup>

A tag define texto de sobrescrito. O texto de sobrescrito aparece meio caractere acima da linha normal, e às vezes é renderizado em uma fonte menor. O texto do superscript pode ser usado para notas de rodapé.

Exemplo:WWW[1]

Tag <address>

A tag define as informações de contato para o autor/proprietário de um documento ou um artigo. As informações de contato podem ser um endereço de e-mail, URL, endereço físico, número de telefone, alça de mídia social, etc. O texto no elemento geralmente é renderizado em itálico, e os navegadores sempre adicionarão uma quebra de linha antes e depois do elemento.

A seguir está um exemplo e uma imagem de como se estrutura uma informação com a tag <address>.

Escrito por Marco Rocha.
Endereço: Travessa São Judas Tadeu, n° 102, bairro São João.
Telefone: (42) 99806-4749

Tag <cite>

A tag define o título de uma obra criativa (por exemplo, um livro, um poema, uma canção, um filme, uma pintura, uma escultura, etc.).

Exemplo:The Scream, obra de Edward Munch, pintadade em 1893.

Tag <pre>

A tag define texto pré-formado. O texto em um elemento é exibido em uma fonte de largura fixa, e o texto preserva espaços e quebras de linha. O texto será exibido exatamente como escrito no código-fonte HTML.
