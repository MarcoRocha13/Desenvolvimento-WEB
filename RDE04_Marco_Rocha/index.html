Tecnologias

As tecnologias utilizadas para o desenvolvimento Web são:

  • HTML 5.0: Uma linguagem de marcação de Hipertexto, responsavel pelo conteúdo do site (texto, tabela e imagem por exemplo).
  • CSS: Responsável pela estilização do conteúdo.
  • Java Script: Reponsável pela interatividade - validação, animação e outras funções.

Página Web ou Webpage é um arquivo onde você pode inserir textos, imagens, tabelas e incorporar planilhas, apresentações, vídeos com ligações de hipertexto que permitem a navegação de uma página, ou seção na outra.

Websites ou Sites são uma coleção de páginas web armazenadas em uma pasta ou em subpastas relacionadas de um servidor web.

HTML

É uma linguagem de marcação de hipertexto baseda em tags criada na década de 1990 por Tim Bernners-Lee para publicação de conteúdo em hipertexto, texto, imagem, audio e video, que podem ser vinculados através de hyperlinks.

As Tags definem ao navegador a marcação de conteúdo. As tags tem funcionalidades específicas. Sem o uso de tags o navegador exibe um texto plano.

As tags podem possuir atributos, ou seja, são parâmetros que definem algum tipo de informação extra ao navegador.

As tags de heading definem o título e possui seis niveis de importância.

As tags de parágrafo definem a apresentação dos parágrafos.

As tags de quebra de linha produzem uma quebra de linha em um texto. É útil para escrever linhas de poemas ou um endereço, onde a divisão de linha é significante.

Imagens

Imagem é uma representação visual de uma pessoa, objeto ou paisagem. Imagem digital é uma representação de uma imagem bidimensional usando números binários codificados de modo a permitir seu armazenamento,transferência, impressão e reprodução e seu processamento por mais meios eletrônicos. Formatos de imagem são padrões de codificação, compreensão e definição que determinam o tipo de arquivo gráfico, tais como .BMP, .TIFF, .GIF, .SVG, .ICO, .JPEG (preserva o fundo), .PNG (fundo transparente) dentre outros

As licenças Creative Commons são várias licenças públicas que permitem a distribuição gratuita de uma obra protegida por direitos autorais.

As tags de imagens e seus parâmetros definem as imagens e suas propriedades.

Marcação de favicon: define um ícone de favoritos que aparece na barra de títulos.
